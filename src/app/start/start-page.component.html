Task

Create small SPA based on Angular 2+ framework.

Description

The test app should contain a users list page and a user details page. For markup you can use any CSS framework or a clear CSS (We use Zurb Foundation). The usage of flexbox will be a plus.
  1. User list page should have a grid component with sort and search functionality.
    • ✔ Sort and search fields should be customizable. The sorting item can be choose by <select>. (Sort name by A-Z, Sort name by Z-A etc.).
    • ✔ Row of the grid should be a separate component and use how a template for grid rows.
    • ✔ After clicking on the row, the user goes to the user details page.
  2. The user details page should have the same information as the grid row.
  3. The users list data should be received from a simple Angular service. As a DB, you can use a static JSON file.
